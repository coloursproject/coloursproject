What are Colours?

Colours is a scraping study of the most popular colours on the web, based on the top 10,000 sites' page elements. Upon release, any person could claim a random colour from the data set for the mint price plus gas. Colours simply represent blockchain code of a specific hex code for a colour in an Ethereum-compatible token format, with attributes attached as metadata in JSON format.

Why should I mint a Colour?

Each NFT represents a colour that has some prestige to it based on which websites use it on the internet. Some colours are especially rare given that they represent a popular brand's identity. You may roll a random rare colour or you may roll a significant website's colour, of which there are only a small number.

Where did you source the top 10,000 sites' colours?

Majestic Million updates its database of top-linked-to websites in the world daily. The .csv file was exported and filtered. The top 10,000 sites were scraped using proprietary code and colours were extracted from page elements to the best of the code's ability.

Why are there 9,839 Colours and not 10,000?

A number of the top 10,000 sites include redundant domains such as "google.de" or "t.co" which redirect to an existing top ranked site. In order to preserve the accuracy of the data set, these were removed during sanitization protecting the integrity of the data set from overrepresentation of a given entity and creating a final sample of 9,839 colours.

How are Colours valued?

Colours are not investment products in any way, shape or form. Colours are representations of real-world colours on the blockchain and their value is derived from whatever people choose to do with them. The owner of a particular colour may choose to do things with their colour that creates or destroys value.

Attributes exist on JSON files on IPFS referenced by the blockchain code that track the rarity of a colour according to how many websites have used the colour and other metrics.

For some significant colours of top websites, the website is assigned to the Colour on the JSON file. There are also a small number of Colours containing alpha channels (8 characters instead of 6) which are also rare.

Who is responsible for Colours?

Colours is a decentralized project with no leadership structure—the initial team simply deploys the code and files for the world. The deployment of the Colours contract and files represents freedom of expression on the blockchain.

439 Colours will be randomly minted to the developer wallet. If any website-specific rare Colours are minted to the developer wallet, they are to be given away to the Discord community to ensure fairness.

How can I build with Colours?

Colours Project aims to be a community-built bottom-up lego project and create infrastructure for on-chain and off-chain Web 3.0 development related projects.

Through the Colours community, you may reach out to a colour owner and inquire about a collaboration. The Colours' hex code will be uploaded to the blockchain directly on a contract function, allowing people to match Token IDs with onchain data outputting the colour.

An example includes combining your Colours Project hex code and N Project numbers and generating an on-chain/off-chain non-random generative art based on the colours and numbers. There is no ceiling on what can be done with your Colours hex code.
